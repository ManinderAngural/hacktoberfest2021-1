# Movie-App  
## Description
A Web App to visualize the top-rated movies.

### Functionalities
> 1.  Quick Search to sort movies.  
> 2. Description of the movie on the hover of the mouse over movie templates.  

### Technologies used
- HTML  
- CSS  
- JavaScript
